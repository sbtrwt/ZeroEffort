# FPS-Shooter

## Code Implementation
- Integrated other asset (cowsin FPS asset)
- State machine for zombie character 
- Singleton design pattern
  
### HOW TO PLAY
- wasd keys for movement
- Mouse for aim the target and camera move
- Click for shoot and interact
- Shift to move fast
